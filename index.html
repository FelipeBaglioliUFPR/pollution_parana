Concentration of PM2.5 in Paraná-Brazil

The map shows PM2.5 concentration for different cities in Paraná, Brazil.

The size of each circle represent the value of the last measurement, also shown in the number inside each circle (ug/m³).

Aditionally, the color of the circles are sorted according to the average concentration measured in each city, as shown on the legend. That means that the size of the circle represents the current state of the air quality. Meanwhile, the color represents the average state of the air quality.

Aditional information can be obtained by clicking on each circle, including the number of days used for average calculation and the date of the last measurement.
